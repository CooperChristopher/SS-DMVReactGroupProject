# reactTraining2

initial setup of project in visual studio code
1. Open visual studio code, and navigate to where you want to place the new project (cd "path")
2. Type git clone https://github.com/SS-Dynamics-August19/SS-DMVReactGroupProject.git
1. open the newly created folder in the react project folder to work with it.
2. open terminal and type npm install.
3. Once previous step is complete, type in terminal npm start.
4. To view the project, go to the url listed near the end of the console after npm start finishes (ex: Server started http://localhost:9090)
